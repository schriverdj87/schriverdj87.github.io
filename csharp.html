Monster Punch

April 2024

A simple game where you click or press space at the right time to launch the fist at oncoming monsters.
This game is a .NET framework windows forms application with separate "presentation"(audio and visuals) and "business" (core logic) layers. In this way, it would be possible to copy the latter over to something like the Unity game engine for a more "gamer friendly" experience.
Either way, this project serves to prove that I can still dust off and make use of technology I haven't used in a while.

    Secondary Technologies: .NET

        7 Fingers

        June 2018

        The aim of this game is to be the number 7. You start at 20 and you move around with WASD keys to make yourself 7 by catching the stylized suit cards that fly across the screen. Red cards (hearts, diamonds) make you decrease while black cards (spades, clubs) make you decrease. If you are 7 by the time the whole deck passes you win. If not or you become too high or too low you lose. If you win, the end screen will show you all the cards you collected. This was made in college for the object-oriented programming final project working with the limitations of windows forms. The name '7 Fingers' comes from a reference to a grizzly story from work safety class I was planning to add but (fortunately) didn’t.

        • Created a card game using sophisticated C# coding and windows forms for an enjoyable user experience.
        • Designed minimalistic cards + backing using photo-manipulation software for a visually appealing look.
        • Recorded video demo and added 'on board' instructions for clarity.
