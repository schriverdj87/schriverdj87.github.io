Flash Cards

2016 - 2018

This program takes plain text and interprets it as quizzes for study purposes. Each line is interpreted as a question and each empty line is interpreted as a divider for a sub quiz. The quizzes can either be made with a text editor and saved to the "bucket" folder or entered into its onboard editor to its SQLite database. This was made before going to college and further developed through "field testing" where I found good use for it. My teacher for Intermediate Java allowed me to develop this for my final project. Please note that the version on GitHub is prior to the SQLite functionality to avoid potential legal issues.

    Neck

    February 2017

    A simple game that puts a twist on the classic snake game. Use the arrow keys to guide Naybur into the sparklies while avoiding skulls and his own neck. This made use of JavaFX's canvas functionality and required extra coding to get that retro look without blurring (anti-aliasing)
