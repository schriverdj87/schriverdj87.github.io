Template Sandwich

 

Create iterative text easily!

  1. Enter any text you want to iterate such as “I am the ” in the top text area.
  2. For the text you want different each time enter “[REP1]”. Another text box should appear below.
    (TIP: To keep things neat you may want to press enter after [REP1])
  3. Type in as many different versions of text as you like separated by commas.
    (TIP: If you want a comma in your text try “,”)
  4. You may add “[REP2]” - “[REP3]” for more complexity.
